# Max Heap and Heap Sort in Python

This task implements a Max Heap data structure and the Heap Sort algorithm in Python. It focuses on demonstrates how heaps work internally and how they can be used to sort arrays in-place.

## Files

- `heap_structure.py`  
  Contains the `MaxHeap` class implementation and a simple demo.

- `heap_sort_algorithm.py`  
  Contains a standalone `heap_sort` function (using an internal `heapify`) and a few test cases.

## How to Run

1. Make sure the following files are in the same directory:
   - `heap_structure.py`
   - `heap_sort_algorithm.py`

2. Run the heap structure demo:

   ```bash
   python heap_structure.py
